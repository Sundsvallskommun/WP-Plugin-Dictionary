This plugin adds dictionary support to post content via a shortcode.

== Description ==

Add words and description of the words in the Dictionary menu item. In post content
you can then add a shortcode [ordlista]my word[/ordlista] or [dictionary]my word[/dictionary] in the text.
On viewing the page it will create a tooltip with the word description when hovering the word. The word will have a
dashed line beneath it.

== Installation ==

1. Upload `wp-plugin-dictionary.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the plugin settings page and set the bootstrap and tether settings as you want.
